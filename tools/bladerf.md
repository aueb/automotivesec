# BladeRF

## Technical Data

* Frequency Range: 300MHz - 3.8GHz
* Independent RX/TX 12-bit 40MSPS quadrature sampling
* Full-duplex 28MHz channels
* 16-bit DAC factory calibrated 38.4MHz +/-1ppm VCTCXO
* USB 3.0
